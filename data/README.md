단지목록 파일을 이 폴더에 넣으면 앱이 시작할 때 자동으로 읽습니다.

권장 파일명:

- `apartments.csv`
- `apartments.json`

CSV 파일은 UTF-8 또는 한글 윈도우 CSV 형식을 사용할 수 있습니다.

필수에 가까운 열:

- `단지코드` 또는 `aptCode` 또는 `kaptCode`
- `단지명` 또는 `kaptName`
- `세대수` 또는 `households`

예시:

```csv
단지코드,단지명,세대수,주소
A51380505,영광대성스카이3차,170,전라남도 영광군
```

파일을 교체한 뒤에는 서버를 다시 시작해야 새 데이터가 적용됩니다.
